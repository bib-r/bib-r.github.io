BIB-R Datasets: T42

T42 has been built for benchmarking FRBRization solutions which deal with MARC (UNIMARC & MARC21) records. The dataset is composed by 42 different tests where each relates a specific FRBR pattern representation and can include specific issues. Browse the dataset T42 on Github. More details below.

Dataset Details

Here you can explore each of the 42 tests from the dataset

1. Core

The category Core concerns a selection of records in which the FRBR pattern consists of a Work, an Expression, a Manifestation and optionally and Agent.

  • 1.0 Basic
  • 1.1 Missing Publication Date
  • 1.2 Missing Record ID
  • 1.3 Missing Type or Form of Material
  • 1.4 Title Linkage Error
  • 1.5 Missing Uniform Title
  • 1.7 Responsibility Linkage Error
  • 1.8 Missing Relator Code
  • 1.9 Missing Authoritative Responsibility
  • 1.10 Cataloguing Practicies

2. Augmentation

The category Augmentation concerns a selection of records where each contain a pattern that leads to the creation of two works where one was used as an additional part of the other.

  • 2.0 Basic
  • 2.1 Missing Publication Date
  • 2.2 Missing Record ID
  • 2.3 Missing Type or Form of Material
  • 2.5 Missing Uniform Title
  • 2.8 Missing Relator Code
  • 2.9 Missing Authoritative Responsibility
  • 2.10 Cataloguing Practicies

3. Derivation

The category Derivation concerns a selection of records where each contains a pattern that leads to the creation of two related works where one is a intellectual evolution of the other.

  • 3.0 Basic
  • 3.1 Missing Publication Date
  • 3.2 Missing Record ID
  • 3.3 Missing Type or Form of Material
  • 3.6 Missing Original Title
  • 3.8 Missing Relator Code
  • 3.9 Missing Authoritative Responsibility
  • 3.10 Cataloguing Practicies

4. Aggregation

The category Aggregation concerns a selection of records where each contains a pattern that leads to the creation of a work related to a set of works.

  • 4.0 Basic
  • 4.1 Missing Publication Date
  • 4.2 Missing Record ID
  • 4.3 Missing Type or Form of Material
  • 4.5 Missing Uniform Title
  • 4.8 Missing Relator Code
  • 4.9 Missing Authoritative Responsibility
  • 4.10 Cataloguing Practicies

5. Complementary works

The category Complementary works concerns a selection of records where each contains a pattern that leads to the creation of related works where each is indenpendant and none is predominant.

  • 5.0 Basic
  • 5.1 Missing Publication Date
  • 5.2 Missing Record ID
  • 5.3 Missing Type or Form of Material
  • 5.5 Missing Uniform Title
  • 5.8 Missing Relator Code
  • 5.9 Missing Authoritative Responsibility
  • 5.10 Cataloguing Practicies
